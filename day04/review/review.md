# 변수

- const[O] / let[종종 쓰셈] / var[x]
- const 변수명 = 데이터(타입에 따라 다름)
- 변수명 짓는 문법 & 국룰
  - 숫자 시작 안됨
  - 띄어쓰기 안됨
  - 중복 안됨
  - 의미있는단어
  - 두글자 이상 합쳐지면 snake,camel

# 데이터 타입

- 기본타입: string, number, boolean, null, undefined
- 참조타입: array, object, window, console
